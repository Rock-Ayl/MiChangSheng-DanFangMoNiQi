# MiChangSheng-DanFangMoNiQi

游戏：觅长生的一个丹方模拟器,可以生成全量的丹方

## 如何使用

- Config.class 包含大部分的配置信息,根据需要修改.
- 枚举类包含小部分配置信息,一般不用修改.
- 运行 Start.class 即可生成全量的丹方

## 考虑并实现的

- 所有丹炉
- 所有药材、妖丹
- 所有单方
- 支持：主药1、主药2、辅药1、辅药2、药引1
- 寒热不平：主药辅药全部满足,药性不平 or 药引药性不够
- 药性相冲：满足丹药1的同时满足丹药2,且不是同一类型丹药
- 丹药等级太高炸炉(三品丹炉炼制六品丹药)
- 无单方的丹药跳过
- 丹方是否包含妖丹
- 高级逻辑【填充物平衡寒热】（问题核心）：
    - 主药2、辅药1、辅药2，可以做为填充物平衡寒热
    - 但是会使得单方生成方案无限大...

## 问题记录

- 单方数量太大了：全部情况都考虑的情况下，丹方组合是千亿级的。
- 单独忽略【主药2格子】情况下
    - 勉强可以生成部分单方...，但还是卡在洗髓丹那里,而且无法导出excel
- 单独忽略【填充物平衡寒热】情况下
    - 可以很快生成全量单方，每个品阶的单方是十万级到百万级
    - 可以导出excel
    - 三品单方超过单个sheet的极限,需要拆分2个sheet

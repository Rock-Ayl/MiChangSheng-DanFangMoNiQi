# MiChangSheng-DanFangMoNiQi

游戏：觅长生的一个丹方模拟器,可以生成全量的丹方

## 如何使用

- Config.class 包含大部分的配置信息,根据需要修改.
- 枚举类包含小部分配置信息,一般不用修改.
- 运行 Start.class 即可生成全量的丹方

# 实际定义的生成规则(已实现)

- 不同品阶丹炉：同一种丹方，如果能用六种品阶的丹炉炼制，算六种丹方。
- 所有药材：同时输出包含妖丹、不包含妖丹，两种情况。
- 所有丹方：排除废丹、四种无丹方的丹药。
- 丹炉格子：完全与丹炉对应，不同丹炉不同考虑，最大支持【主药1、主药2、辅药1、辅药2、药引1】
- 排除寒热不平：主药辅药全部满足的情况下,寒热不平 or 药引药性不够
- 排除药性相冲：满足丹药1的同时满足丹药2,且不是同一类型丹药
- 丹药等级太高炸炉(三品丹炉炼制六品丹药)
- 特殊逻辑【填充物平衡寒热】（也是产生大量丹方的核心原因）：
    - 主药2、辅药1、辅药2，如果基础丹方不需要，可以做为【填充物平衡寒热】。
    - 著名的竹山宗启灵丹就是如此，金虹六品丹炉无限炼制强魂丹也是这个原理。
    - 但是这会使得生成方案呈指数级增加...（大概的意思是从 159 * 159 * 159 变成了 159 * 159 * 159 * 159 * 159）

## 问题记录

- 数量太大了：全部情况都考虑的情况下，丹方组合是千亿级的。
- 单独忽略【主药2格子】情况下
    - 勉强可以生成有效...，但还是卡在洗髓丹那里超时,而且无法导出excel
- 单独忽略【填充物平衡寒热】情况下
    - 可以很快生成全量有效，每个品阶的是十万级到百万级
    - 可以导出excel
    - 三品超过单个sheet的极限,需要拆分2个sheet
- 即使只考虑【辅药2】作为【填充物平衡寒热】
    - 随意一个一品丹方、一个丹炉，都有30万的有效丹方。
    - 能导出，但是只能是txt文档了，失去了表格的意义。
